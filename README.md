# ASCII Arch Tarot Logo for Fastfetch

I was looking for an ASCII tarot card for fastfetch, and decided to make my own!

Make sure to create a path within the config.jsonc file to the .txt file.
The card also requires a height of 28, which can be specified in the config.jsonc.
